# Teacup 0.2.0
### Description
Library based on pysdl2 designed for building windows applications quickly.
Refer to the "examples" folder for usage.

### Installation
1. Navigate to the releases page and find the latest version (it should be at the top)
2. Download the .zip file and decompress it.
3. Open terminal of choice and navigate to the unziped folder
4. Run `pip install .`
5. To verify installation run `pip list` and check if "teacup" is in the list
